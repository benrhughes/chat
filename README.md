## Build
```
git clone https://github.com/benrhughes/chat.git
cd chat
npm install
npm run build
```

Serve the contents of the `dist` folder, or you can simply download the contends of the [gh-pages](https://github.com/benrhughes/chat/tree/gh-pages) branch.
